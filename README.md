# naomisugar-electron

An Electron wrapper over https://github.com/vpotravnyy/naomisugar that shows it in MacOS tray menu

## Project Setup

### Install

```bash
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

### Build

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
